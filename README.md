# FIN-5 金融职业匹配测试（静态网站）

一个“类似 MBTI 的金融职业匹配”小测试：25 题、5 个维度（FIN-5），给出你的类型与职业匹配 Top 3。

## 快速使用

1. 直接用浏览器打开 `index.html` 即可运行（纯前端、无需安装依赖）。
2. 如果你希望通过本地服务器访问（可选）：
   - 若你装了 Python：在本目录运行 `python -m http.server 8000`，然后访问 `http://localhost:8000`

> 说明：页面只在本地运行，不会上传你的答案。
> 默认每次打开都会清空上次作答（不做浏览器本地保存）。

## 文件结构

- `index.html`：页面结构（介绍 / 做题 / 结果）
- `styles.css`：样式
- `data.js`：题目、维度、职业库（含每个职业的 FIN-5 “目标向量”）
- `app.js`：渲染与打分逻辑（向量归一化、职业匹配、结果生成）
- `questions.md`：题库的可读版本（方便改题）

## 如何改题 / 改职业

- 改题：编辑 `data.js` 里的 `FIN5_QUESTIONS`
  - 每题有 `dim`（所属维度）+ `left/right`（两侧陈述）
  - 选项值范围是 **-2（强偏左）到 +2（强偏右）**，会按维度归一化到 [-1, 1]
- 改职业：编辑 `data.js` 里的 `FINANCE_CAREERS`
  - `vector` 是该职业在 5 个维度上的“典型倾向”（范围 [-1, 1]）
  - 匹配度是把你的向量与职业向量做距离计算后换算为 0–100 分
