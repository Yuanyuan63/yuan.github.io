# FIN-5 金融职业匹配题库（25题）

FIN-5 以 5 个“二元维度”刻画你的偏好，每题在两种倾向之间选择程度（偏左/中立/偏右）。

## 五个维度

1. **S vs L（节奏）**
   - S：快节奏/短反馈/临场执行
   - L：长周期/深度打磨/耐心迭代
2. **C vs A（对客）**
   - C：对外沟通/客户与协作驱动
   - A：对内产出/分析与内容驱动
3. **R vs P（风险）**
   - R：进攻性/愿意承担不确定性换收益
   - P：审慎性/偏好可控与稳定
4. **Q vs B（技能形态）**
   - Q：数理/数据/代码/模型
   - B：商业/表达/谈判/叙事
5. **O vs X（工作取向）**
   - O：规则/流程/治理/可审计
   - X：创新/开拓/产品与策略探索

## 题目（每题：左倾向 / 右倾向）

### S–L（节奏）
1. 更喜欢短反馈周期（当天/当周看到结果） / 更喜欢长周期投入（数月打磨更大成果）
2. 享受实时变化带来的刺激（行情/突发信息） / 享受稳定推进的深度工作（调研/项目）
3. 倾向于多任务快速切换 / 倾向于少量关键任务持续深入
4. 更偏好“先执行、边做边校准” / 更偏好“先打磨框架、再稳步落地”
5. 更愿意在强时点压力下冲刺（截止日、开盘收盘） / 更愿意在可控节奏里长期迭代

### C–A（对客）
6. 更愿意站在台前对外沟通、影响他人 / 更愿意在台后做分析产出，让结果替我说话
7. 从社交与协作中补能 / 从独立专注中补能
8. 更喜欢把复杂内容讲清楚并推动决策 / 更喜欢把复杂内容拆解清楚并做成模型/文档
9. 更喜欢开放式问题：先沟通需求再给方案 / 更喜欢结构化问题：定义假设并验证
10. 更在意建立长期客户/合作关系 / 更在意把内部能力（研究/系统/流程）做强

### R–P（风险）
11. 为更高回报能接受明显波动与不确定性 / 更重视可预测性，宁愿牺牲部分收益
12. 更愿意对“结果”负责（收益/增长） / 更愿意对“底线”负责（风险/合规/稳定）
13. 做决策时更愿意先下注再迭代 / 做决策时更愿意先把风险边界想清楚
14. 不介意在信息不完美时做判断 / 更倾向在证据充分时再行动
15. 更认可“进攻性目标”的驱动力 / 更认可“少出事/不出事”的安全感

### Q–B（技能形态）
16. 更享受用数据/统计/数学把问题讲清楚 / 更享受用商业逻辑/故事线把问题讲清楚
17. 更愿意写代码、做自动化、搭模型 / 更愿意写材料、谈判、对齐共识
18. 遇到问题更倾向先查数据、跑实验 / 遇到问题更倾向先访谈、做判断
19. 更喜欢“可复现的证据链” / 更喜欢“有说服力的论证与表达”
20. 对工程细节（性能、稳定性、数据质量）更敏感 / 对业务细节（定位、需求、利益相关方）更敏感

### O–X（工作取向）
21. 更喜欢明确规则、标准流程、可审计 / 更喜欢探索新方法、新产品、模糊边界的机会
22. 更愿意做“把事情做对”的角色（控制/检查/治理） / 更愿意做“把事情做成”的角色（创新/开拓）
23. 更享受把流程优化得更稳、更少差错 / 更享受把方案迭代得更快、更有突破
24. 面对灰度地带更倾向保守解释与留痕 / 面对灰度地带更倾向试点验证与快速学习
25. 更愿意在既有规则内做增量改进 / 更愿意推动规则/产品/策略的显著变化
